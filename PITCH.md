# Agente InHire — Pitch

## O Problema
Recrutadores gastam 70% do tempo em tarefas repetitivas: triagem de curriculos, agendamento de entrevistas, hunting manual no LinkedIn. Sobra pouco tempo pra o que realmente importa: avaliar pessoas.

## A Solucao
Um agente de IA integrado ao Slack que funciona como um assistente de recrutamento:
- Cria vagas conversando em linguagem natural
- Faz triagem e ranking de candidatos automaticamente
- Agenda entrevistas coordenando com candidatos
- Faz hunting ativo buscando perfis compatveis
- Aprende com o feedback do recrutador

## Como Funciona
1. Recrutador abre o Slack e diz: "Preciso de um dev Python senior"
2. Agente cria a vaga na InHire com os requisitos
3. Candidatos sao triados e rankeados automaticamente
4. Agente sugere os top 5 e agenda entrevistas
5. Proactive Monitor avisa quando algo precisa de atencao

## Diferenciais
- Zero troca de contexto (tudo no Slack)
- Aprende com cada interacao
- Chrome extension pra hunting no LinkedIn
- Integracao nativa com a API InHire

## Stack
Python, FastAPI, Claude API, Slack Bot, Chrome Extension
